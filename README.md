# Jogo da Forca

## Descrição
 
 Este é um jogo da forca de console simples. O objetivo do jogo é adivinhar a palavra secreta antes de atingir o número máximo de erros permitidos.

## Funcionalidade
 
 - **Escolha Aleatória de Palavras**:  O jogo seleciona uma palavra de forma aleatória de um conjunto predefinido.
 - **Interface Intuitiva Via Console**: O jogo exibe as informações de forma clara e objetiva no terminal.
 - **Sistema de Tentativas**: Contabiliza o número de erros permitidos e finaliza o jogo ao atingir o limite.
 - **Exibição do Progresso**: Mostra a palavra parcialmente revelada com base nas letras acertadas, e mostra os erros por meio de mensagem e do desenho da forca.
 - **Finalização do Jogo**: O jogo exibe uma mensagem de vitória ao completar a palavra ou de derrota ao atingir o limite de tentativas.
